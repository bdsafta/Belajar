Cara Backup Project Local Ke Github

# 1. Masuk ke folder project PHP (jika belum)
cd path/ke/projectmu

# 2. Cek status perubahan
git status

# 3. Tambahkan semua perubahan
git add .

# 4. Commit dengan pesan
git commit -m "Update beberapa kode pada project PHP"

# 5. Push ke GitHub (sesuaikan branch: main atau master)
git push origin main
